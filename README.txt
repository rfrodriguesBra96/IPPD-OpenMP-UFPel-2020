	Como executar o programa:
 1 - Abra seu terminal.
 2 - Digite: make
 
 	Como executar os inputs:
./nbody_simulation input_files/nbody_input-16384_16384.in <número de threads>
./nbody_simulation input_files/nbody_input-32768_32768.in <número de threads>
./nbody_simulation input_files/nbody_input-65536_65536.in <número de threads>
./nbody_simulation input_files/nbody_input-131072_131072.in <número de threads>

	Exemplo:
> ./nbody_simulation input_files/nbody_input-16384_16384.in 4
Executa o input que usa 16384 particulas com 4 threads.

	Conteúdo do diretório
Input_files: pasta que contém os arquivos(base de dados).
